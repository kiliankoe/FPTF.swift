FPTF

This package includes the necessary types for working with the friendly-public-transport-format. All types conform to Swift’s Codable protocol to enable seamless de- and encoding of FPTF-JSON.

Installation

FPTF is available via Swift Package Manager, just add it to your project’s dependencies.

.package(url: "https://github.com/kiliankoe/FPTF", from: "latest version")
